# Drawing a line

![white line](https://threejs.org/examples/screenshots/webgl_test_wide_gamut.jpg)

## You can draw lines

You can draw a line pretty easy. You need to use `LineBasicMaterial` / `LineDashedMaterial`:

```javascript
const material = new THREE.LineBasicMaterial({ color: 0x0000ff });
```

We also need a geometry with some vertices:

```javascript
const points = [];
points.push(new THREE.Vector3(-10, 0, 0));
points.push(new THREE.Vector3(0, 10, 0));
points.push(new THREE.Vector3(10, 0, 0));

const geometry = new THREE.BufferGeometry().setFromPoints(points);
```

Now that we have points and a material for the line, we can add them together - then add it to the scene:

```javascript
const line = new THREE.Line(geometry, material);

scene.add(line);
```

---

This is what the overall code should look like to create a blue arrow:

```javascript
const points = [];
points.push(new THREE.Vector3(-10, 0, 0));
points.push(new THREE.Vector3(0, 10, 0));
points.push(new THREE.Vector3(10, 0, 0));

const geometry = new THREE.BufferGeometry().setFromPoints(points);
const material = new THREE.LineBasicMaterial({ color: 0x0000ff });
const line = new THREE.Line(geometry, material);

scene.add(line);
```
