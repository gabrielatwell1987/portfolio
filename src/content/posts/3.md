# Create text

![text](https://threejs.org/examples/screenshots/webgl_modifier_tessellation.jpg)

## using the dom

The fastest way to add text is through html and to style that text you can use css.

<div style="font-size: .75rem">html</div>

```html
<div id="description">You text goes here</div>
```

<div style="font-size: .75rem">css</div>

```css
#description {
    position: absolute;
    top: 10px;
    display: block;
    inline-size: 100%;
    text-align: center;
    z-index: 100;
}
```

### Procedural Text

If you to create procedural and dynamic 3D text, you can create a THREE.TextGeometry: `new THREE.TextGeometry( text, parameters );`

[TextGeometry](https://threejs.org/docs/?q=TextGeom#TextGeometry)
